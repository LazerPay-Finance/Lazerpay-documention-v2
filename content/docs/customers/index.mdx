# Customers

Keep track of all your customers with this API endpoint you can programmatically fetch and list all the customers linked to your Lazerpay account.

## All Customers

List all the transactions associated with your Lazerpay account. 

<br />

<CodeBlock
    responseTheme="default"
    responseTitle="Response"
    data={{
        type: 'request',
        item: {method: "POST", requestData: {
            name: ['cURL', 'Javascript'], 
            snippet: ['allCustomersPostCurl', 'allCustomersPostJs'], 
            lang: ['curl', 'js']
        }, 
        responseData: {
            name: ['200: OK', '400: Bad Request'], 
            snippet: ['allCustomers200', 'allCustomers400'], 
            lang: ['json', 'json']
        }} 
    }}
/>
